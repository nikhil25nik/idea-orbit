# IdeaOrbit 

IdeaOrbit is a full-stack blog application built using the MERN stack.  
It allows users to create, read, update, and delete blog posts with secure authentication.

##  Features
- User Authentication (Signup / Login)
- aslo use firebase for google login
- Create, Edit & Delete Blog Posts
- View All Blogs & Individual Blog Pages
- Responsive UI using Shadcn ui
- Secure REST APIs
- MongoDB Database Integration

## Tech Stack
- **Frontend:** React.js,Shadcn Ui, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** bcryptjs / JWT, firebase
- **Tools:** Git, GitHub, VS Code

##  Project Structure
IdeaOrbit/
│── client/ # React Frontend
│── api/ # Node + Express Backend
│── models/ # MongoDB Schemas
│── routes/ # API Routes
│── controllers/ # Logic
│── utils/ # Helpers & Middleware


## How to Run Locally

### Clone the Repository
git clone https://github.com/your-username/IdeaOrbit.git
cd IdeaOrbit

## Backend Setup
cd api
npm install

## Run backend:
nodemon index.js

## Frontend Setup

cd client
npm install
npm run dev

## Purpose

IdeaOrbit was built to:
